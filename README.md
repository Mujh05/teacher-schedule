# 英语单词排课管理系统

面向线上英语单词教学老师的排课管理工具。

## 功能特性

### 课程管理
- **新建排课**：选择学生、星期（支持多选）、时间（50分钟/节），填写学习内容和教材版本，设置起止日期（过期自动隐藏）
- **编辑/删除排课**：点击课程卡片编辑或删除
- **排课有效期**：可设置开始/结束日期，仅在日期范围内的周次显示，结束日期留空表示长期有效。默认开始日期为当天所在周的周一
- **体验课转正**：给体验课学生排课时自动提示是否转固定，转正后该学生所有非体验课触发 ¥30/节的报名奖励

### 学生管理
- **分类管理**：固定、体验、临时、代课四种分类
- **学生信息**：姓名、分类、年级、联系方式、备注
- **状态同步**：学生分类变更时标记 wasTrial 永久记录，已有排课保持原分类不变

### 多种视图
- **周视图**（默认）：显示当前周课程表，今日高亮，左右切换周
- **月视图**：日历格式显示每天课程数，点击某天跳转到对应周视图
- **列表视图**：表格形式展示所有排课，支持编辑/删除
- **学生视图**：按学生分组展示排课

### 课程状态标记
- **⏳ 待命**：有效待命状态，表示课程已约定但未确认

### 收入统计
- **课时费**：每节正常课 ¥30
- **待命费**：每节待命课 ¥10
- **报名奖励**：从体验课转正的学生，每节非体验课额外 ¥30，显示奖励节数
  - 奖励条件：学生必须先上过体验课，再转为固定/临时/代课
  - 统计所有排课（正常课 ¥30 + 待命课 ¥10）
- **每月收入趋势折线图**：可视化展示月度收入变化
- **每周各天排课数柱状图**：直观查看课程分布
- **有效待命数**：实时统计

### 其他功能
- **数据持久化**：localStorage + 文件双写（Electron 版存储到 `{userData}/schedule-data/schedule_data.json`），刷新不丢失
- **深色模式**：支持明暗主题切换
- **数据导入导出**：支持 JSON 格式导入导出
- **课前提醒**（Electron 桌面版）：支持 Windows 原生通知，可设置提前 0~15 分钟提醒，每30秒自动检查课程时间

## 使用方法

直接双击 `教师课程表.html` 用浏览器打开即可使用。

### Electron 桌面版

```bash
cd electron-app
npm install
npm start          # 开发运行
npm run build      # 打包成安装程序
```

## 数据结构

```
db.students[]    → 学生列表
  - id, name, category(固定/体验/临时/代课), grade, phone, note, createdAt

db.schedules[]   → 排课列表
  - id, studentId, category, dayOfWeek(0-6), startTime, endTime,
    content(学习内容), textbook(教材版本), note, standby(是否待命),
    startDate(生效日期), endDate(截止日期)

reminderConf   → 提醒设置（localStorage: eng_reminder）
  - advanceMinutes(提前分钟数: 0/1/2/3/5/10/15, 默认5)
```

## 技术栈

- 纯 HTML + CSS + JavaScript，无任何外部依赖
- 单文件架构，一个 `.html` 文件即为完整应用
- localStorage 本地存储
- Electron 桌面打包（可选）
